ウレシイお知らせです。  
  
平成１３年　文化庁メディア芸術祭  
デジタルアート　インタラクティブ部門において  
平沢進「インタラクティブ・ライブショウ2000 賢者のプロペラ」が、「審査委員会推薦作品」に選出されました。  
  
また、平沢が音楽を手掛けたアニメ「千年女優」は、  
なんとアニメ部門の大賞を受賞致しました。  
「千と千尋の神隠し」との、異例のダブル受賞  
です。  
  
これらは、以下の会場でも展示が決定しております。  
第5回文化庁メディア芸術祭受賞作品展  
会期：2002年3月1日～10日  
会場：東京都写真美術館
